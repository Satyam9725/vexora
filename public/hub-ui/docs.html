What is Vexora?

Vexora is a modern Node.js framework that provides the essential features needed to build web applications, APIs, real-time applications and more. It is designed with simplicity, flexibility and performance in mind.

Terminal 📋
$ npm install vexora
# or
$ yarn add vexora
# or
$ pnpm add vexora
